# Privacy Policy for Flow

Effective date: March 27, 2026

Publisher: LumoRez

Contact: https://github.com/lumorez07/flow/issues

This Privacy Policy explains how Flow collects, uses, stores, and shares information when you use the Flow desktop application on Windows.

Flow is a teleprompter application with optional voice features, optional AI-assisted text generation, optional remote messaging, and optional update and model download features. Some features work entirely on-device, while others contact third-party services only when you choose to use them.

## What Flow collects

Flow may process the following categories of information:

- Script content you type, paste, import, edit, or save in the app.
- App settings and preferences, such as theme, language, playback mode, window placement, remote settings, and voice settings.
- Optional API credentials you choose to enter, such as a Groq API key for AI text generation.
- Optional remote messaging data, such as receiver session ID, access password, message title, message content, and delivery metadata.
- Optional microphone audio used for voice tracking or voice commands.
- Optional locally downloaded voice model files and related install status.
- Standard network metadata that may be generated when online features are used, such as IP address, time of request, and basic device or browser request headers handled by the services you connect to.

## How Flow uses information

Flow uses information to:

- Display and run the teleprompter.
- Save your script and preferences locally on your device.
- Enable optional voice tracking and voice commands.
- Enable optional AI-assisted script generation or rewriting.
- Enable optional remote message delivery to a running Flow receiver.
- Download optional speech models and check for optional installer updates.
- Maintain app functionality, security, and reliability.

## Data stored locally on your device

Flow stores most user data locally on your device. This may include:

- Script text.
- App preferences and UI state.
- Groq API key and AI prompt preferences.
- Remote receiver identifiers and access settings.
- Downloaded offline voice model archives.

This local data is stored to provide app functionality and convenience. You can remove this data by clearing app data, changing your settings, or uninstalling the app.

## Microphone and voice processing

If you enable voice tracking or voice commands, Flow requests access to your microphone.

- Microphone access is optional.
- Voice processing for teleprompter features is designed to run locally on-device using bundled or downloaded speech recognition models.
- Flow does not send microphone audio to the developer's servers as part of its normal voice feature operation.

If you do not want microphone access, do not enable voice features or deny microphone permission in Windows.

## Optional AI features

Flow includes optional AI-assisted writing features that use Groq only when you choose to use them.

When you use these features, Flow may send the following directly to Groq:

- The API key you provided.
- The instruction or prompt you entered.
- The script text or text segment you asked Groq to generate, rewrite, or transform.

This processing is performed by Groq under Groq's own terms and privacy practices, not solely under this Privacy Policy. If you use Groq features, you are responsible for reviewing Groq's applicable policies.

## Optional remote messaging

Flow includes an optional remote messaging feature. If you enable and use this feature, the following may be processed to deliver messages:

- Receiver session ID.
- Access password.
- Message title and content.
- Message importance flag.
- Sender IP address or related connection metadata used for delivery, rate limiting, and abuse prevention.

Remote messaging may use cloud-hosted infrastructure and public internet routing. You should not use remote messaging to transmit sensitive personal data unless you are comfortable with that risk.

## Optional downloads and online requests

Flow may make network requests in the following cases:

- When you download optional voice recognition models.
- When you use optional Groq AI features.
- When you use optional remote messaging.
- When the app checks for available installer updates.
- When the app loads optional document import helper libraries used for PDF or DOCX import support.

These requests may involve third-party services or infrastructure providers. Those services may receive standard technical metadata, such as IP address, timestamps, and user agent or request information.

## Third-party services and infrastructure

Depending on the features you use, Flow may interact with third-party services or infrastructure such as:

- Groq, for optional AI text generation or rewriting.
- Cloud hosting used for remote messaging.
- GitHub release or installer hosting used for update checks or downloads.
- Content delivery networks used to load optional PDF or DOCX import libraries.
- Hosting used for optional voice model downloads.

Each third-party service operates under its own terms, privacy policy, and security practices.

## What Flow does not do

Based on the current version of the app:

- Flow does not require account registration to use its core teleprompter functions.
- Flow does not include in-app advertising.
- Flow does not sell your personal data.
- Flow does not use built-in analytics or telemetry SDKs for general user tracking in the current version.

## Data sharing

Flow shares data only as needed to provide optional features you choose to use, such as:

- Sending prompts and script text to Groq when you request AI output.
- Sending remote messaging data through relay infrastructure when you use remote delivery.
- Contacting download or update hosts when you request or allow those operations.

Flow does not share your data with third parties for advertising resale.

## Data retention

- Local app data remains on your device until you delete it, clear app data, or uninstall the app.
- Remote messaging data may be retained temporarily for message delivery, abuse prevention, logging, or troubleshooting, depending on the infrastructure used.
- Third-party providers may retain data according to their own retention policies.

## Security

The developer takes reasonable steps to reduce privacy and security risks, but no software or transmission method is completely secure. You are responsible for protecting your device, local account, and any API keys or access passwords you choose to store in the app.

Do not store highly sensitive secrets in the app unless you accept the risks associated with local storage and third-party service use.

## Children's privacy

Flow is not directed to children under 13, and the developer does not knowingly collect personal information from children under 13 through the app.

## International data transfers

If you use online features, your data may be processed in countries other than your own, depending on the third-party services or hosting providers involved.

## Your choices and controls

You can limit or control data processing by:

- Using Flow only in offline or local-only modes.
- Not enabling microphone-based features.
- Not entering a Groq API key.
- Not using remote messaging.
- Not downloading optional voice models.
- Removing local app data or uninstalling the app.

## Changes to this Privacy Policy

This Privacy Policy may be updated from time to time. The updated version will be posted with a new effective date.
