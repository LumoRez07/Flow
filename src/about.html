About this project

Flow 1.1.0

A modern desktop teleprompter for smooth reading, quick editing, voice controls, and remote message injection.

Flow is a teleprompter app built with web technologies and Tauri. It's designed to be simple, lightweight, and customizable.

This project is open source and available on my GitHub account. if you have any questions, suggestions, or want to contribute, feel free to reach out or open an issue.

This project was made by LumoRez with ❤️ in 2026.

Flow includes script editing, multiple playback modes, voice tracking, AI-assisted drafting, remote notifications, tray controls, and Windows-first privacy options like capture protection.
